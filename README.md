## 🌈 Service Introduce

수어지교는 학습·대화·게임 3가지 기능을 통해 수어를 쉽고 재미있게 익힐 수 있도록 설계된 통합 수어 학습 플랫폼입니다.
리듬 게임이라는 게이미피케이션을 접목하여 학습의 지루함을 줄이고, 지속 가능한 수어 학습 환경을 제공합니다.

### 주요 기능

1. 수어 기초 학습(자음, 모음 등) 영상과 설명 학습 기능

2. 단어, 문장을 Daily 과제 형식으로 진행
     (진행률은 데일리 과제 수행 기준)

3. AI와의 대화 기능. 사용자가 원하는 문장을 수어 문법에 맞게 가이드라인 제공

4. 리듬 게임 기능.
 -  타이밍에 맞는 수어를 진행하고 해당 동작이 제대로 되어있는지 동작유사도 모델을 사용하여 판별

5. 마이페이지 기능
- 회원정보 수정, 탈퇴 기능
- 학습 현황 확인

## 🛠️ 기술 스택

- **SpringBoot** 3.5.3
- **MySQL** 8.0
- **Redis** 8.0
- **FastApi**
- **Jenkins**
