- **사용 위치**: AI 서버 (FastAPI)

### 1.4 가입 및 설정 방법

#### Step 1: GMS 계정 확인
1. gms.ssafy.io/web/ 이동
2. ssafy 계정으로 로그인
3. GMS API KEY 확인

#### Step 2: API 키 복사
1. 화면 중앙의 api key를 복사
2. .env 파일에 GMS_KEY='gms api key' 입력 후 저장ㄹ

#### Step 3: 사용량 제한 설정
1. AI_server / app / services / chatbot.py
2. chatting 함수의 payload max_tokens 확인
3. 원하는 토큰으로 제한 설정

### 1.5 API 사용량 및 최적화

#### 토큰 사용량 (예상)
- **정보 제공 요청**: 평균 500-1000 토큰/요청

### 1.6 예상 비용 (월간)
- **GPT-4.1-mini**: $0.000150/1K 토큰 (입력), $0.000600/1K 토큰 (출력)
- **예상 월간 비용**
    - (1) 입력 100만 토큰, 출력 100만 토큰  
        입력: 1,000,000 ÷ 1,000 × $0.000150 = $0.15  
        출력: 1,000,000 ÷ 1,000 × $0.000600 = $0.60  
        합계: $0.75 / 월

    - (2) 하루 10만 토큰 사용 (입력+출력 합)  
        하루: 100,000 ÷ 1,000 × 평균 단가($0.000375, 입력과 출력 1:1 가정) ≈ $0.0375  
        한 달(30일): $0.0375 × 30 = $1.13 / 월

    - (3) 하루 100만 토큰 사용 (입력+출력 합)  
        하루: 1,000,000 ÷ 1,000 × $0.000375 = $0.375  
        한 달: $0.375 × 30 = $11.25 / 월
