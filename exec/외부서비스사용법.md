## 1. OpenAI API 서비스 (AI 기능)

### 1.1 서비스 개요
- **목적**: AI 기반 수어 문법 정보 제공
- **제공업체**: OpenAI
- **사용 위치**: AI 서버 (FastAPI)

### 1.2 사용 모델 및 API
```python
# 사용 모델 목록

    "정보 제공": "gpt-4.1-mini", 
}
```

### 1.4 가입 및 설정 방법

#### Step 1: OpenAI 계정 생성
1. [OpenAI Platform](https://platform.openai.com/) 접속
2. 계정 생성 및 전화번호 인증
3. 결제 정보 등록 (크레딧 충전)

#### Step 2: API 키 발급
1. **API Keys** 탭 이동
2. **Create new secret key** 클릭
3. 키 이름 지정 (예: debate-arena-production)
4. 권한 설정: **All** 또는 필요한 권한만 선택
5. 생성된 키를 안전하게 보관

#### Step 3: 사용량 제한 설정
1. **Usage limits** 탭에서 월간 사용량 제한 설정
2. **Soft limit**: $50 (알림 발송)
3. **Hard limit**: $100 (API 자동 차단)

### 1.5 API 사용량 및 최적화

### 1.6 예상 비용 (월간)
- **GPT-4.1-mini**: $0.000150/1K 토큰 (입력), $0.000600/1K 토큰 (출력)
- **예상 월간 비용**
    - (1) 입력 100만 토큰, 출력 100만 토큰  
        입력: 1,000,000 ÷ 1,000 × $0.000150 = $0.15  
        출력: 1,000,000 ÷ 1,000 × $0.000600 = $0.60  
        합계: $0.75 / 월

    - (2) 하루 10만 토큰 사용 (입력+출력 합)  
        하루: 100,000 ÷ 1,000 × 평균 단가($0.000375, 입력과 출력 1:1 가정) ≈ $0.0375  
        한 달(30일): $0.0375 × 30 = $1.13 / 월

    - (3) 하루 100만 토큰 사용 (입력+출력 합)  
        하루: 1,000,000 ÷ 1,000 × $0.000375 = $0.375  
        한 달: $0.375 × 30 = $11.25 / 월
