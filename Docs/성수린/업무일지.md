### 2\. 수어 인식 모델 조사

- 수어는 손 표현 뿐만 아니라, 손의 위치, 얼굴 표정을 모두 사용해서 표현하는 언어임. 따라서, 손의 표현, 손의 위치를 좌표로 뽑고, 얼굴 표정을 통해 감정 인식을 하여 표현하고자 하는 것을 잘 전달할 수 있도록 하는 것이 중요함.
- **MediaPipe** 또는 **OpenPose**와 같은 좌표를 추출해주는 모델을 통해 손의 좌표, 팔의 좌표, 얼굴 좌표를 추출함.
- **CNN + LSTM** 또는 **Yolo** 모델을 통해 좌표를 시간 순으로 인식해서 어떤 말을 표현하는 것인지를 파악함.
- 대화 기능의 경우, **LLM 모델**에게 수어를 텍스트로 번역한 것을 전달하고, 알맞은 답변을 받음. 이때, 답변을 받은 것을 수어로 잘 표현할 수 있도록 텍스트의 순서를 변환하는 과정이 필요함. 변환 후, 텍스트에 맞는 수어를 보여줄 수 있도록 3D 렌더링함.
    - LLM 모델의 경우에도, API를 사용하는 것이 아닌 **직접 모델을 서빙**해서 사용할 예정.
- 게임 기능의 경우, 좌표 인식 모델 + 좌표를 통한 수어 인식 모델을 통해 알맞은 동작을 했는지 하지 않았는지 판단함.
